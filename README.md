# 2026-shion-portfolio
##　概要
今までの作品をまとめたポートフォリオサイトです。  
HTML/CSS/JavaScriptを使用したフロントエンドと、  
PHPでのバックエンドとの連携を学習するために作成しました。

[ポートフォリオサイトへ](https://shion-portfolio.infinityfreeapp.com/)

##  作品詳細
### お問い合わせフォーム
JavaScriptとPHPの学習を目的として作成しました。  
PHPを初めて開発に使用したことと、まだまだ学習中のJavaScriptを使用したこともあり、最も困ることが多かった作品です。素早くできるPHPの動作確認の仕方や、フロントからPHPにデータを送り、サーバー側で処理される流れなど、多くを学ぶことができました。header.phpやfooter.phpを作って共通部分を独立させることで、変更を1箇所で行えるようにして効率を上げるなどの工夫もしました。
### プロフィールサイト
最初に自分で設計して作り上げた作品です。  
私がデザインを元に実際にそれを形にしていくウェブサイト制作の楽しさに気づいたきっかけはこの作品です。デザインを特にこだわり、レスポンシブ対応にも力を入れました。技術面ではCSSを使ってハンバーガーメニューを作ったり、アニメーションを作ったりなど、JavaScriptを使わないことでサイトのパフォーマンスを向上させることに興味がありました。
### JavaScriptを使った機能集
機能集なので別々のページに分けるのではなく、同じ画面内で切り替えられるようにすることでスムーズにストレスなく各機能を確認できるようにしました。
